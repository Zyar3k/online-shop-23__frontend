# EzzyShop online-shop

> Online store created from scratch. Application server and client. The ability to add products to your cart. Admin dashboard

##### Used in project

- next.js
- next-auth
- next-connect
- react
- chakra
- sass
- bcryptjs
- crypto
- yup

### Installation

To run app

```sh
npm i / yarn install
npm run dev / yarn run dev
```
